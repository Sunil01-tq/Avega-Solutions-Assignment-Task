Added README for manual test cases

 Overview
This repository contains manual test cases designed for validating the core functionalities of the application and Transactions of Firefly III functionalities.

The test cases are created in Excel format and include Test description, steps, expected results, and different test scenarios.

 Test Coverage
The test cases cover:

- Positive scenarios (valid login)
- Negative scenarios (invalid credentials)
- Boundary value Analysis for OrangeHRM salary submoduleslate
- Positive and Negative scenarios for Firefly III Transactions page 
- Field validations 

 Test Case Details
Each test case includes:

- Test Case ID
- Test Description
- Test Steps
- Test Data
- Expected Result
- Priority
- Status (Pass/Fail)

 Objective
To ensure the login functionality works correctly under various input conditions and validates proper error handling.
